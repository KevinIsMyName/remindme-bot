# remindme-bot

## Quick start

1. Create `.env` with keys for...
   1. PREFIX
   2. CLIENT_ID
   3. CHANNEL_ID
   4. SERVER_ID
   5. PUBLIC_KEY
   6. APP_ID
   7. TOKEN
2. Install [discord.py](https://pypi.org/project/discord.py/), [pytimeparse](https://pypi.org/project/pytimeparse/), and [dotenv](https://pypi.org/project/python-dotenv/) via pip.
3. Execute `bot.py`
